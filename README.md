# ButtonForge Classic

A modern recreation of the original ButtonForge addon for Vanilla World of Warcraft (1.12), designed specifically for Turtle WoW and other Vanilla-based servers.

ButtonForge Classic lets you create fully customizable action bars anywhere on your screen while maintaining the look and feel of the original Blizzard interface.

## Features

### Custom Action Bars

* Create unlimited custom action bars (within available Vanilla action slots)
* Freely move and position bars
* Adjustable rows and columns
* Adjustable scale
* Lock and unlock bars
* Show or hide bar backgrounds
* Automatically hide empty buttons outside of configuration mode

### Drag & Drop

* Drag and drop spells directly from the spellbook
* Drag items from your bags
* Drag macros from the macro window
* Swap buttons by dragging between slots
* Move actions to empty slots
* Stable internal drag-and-drop system optimized for Vanilla

### Keybinding

* Built-in Keybinding Mode
* Visual keybinding overlay
* Supports:

  * Keyboard keys
  * Mouse Wheel Up / Down
  * Middle Mouse (MM)
  * Mouse Button 4 (M4)
  * Mouse Button 5 (M5)
* Left and Right Mouse Buttons are protected to prevent accidental camera or interaction issues
* Keybindings remain attached to button positions
* Character-specific keybinding profiles

### Blizzard-style Action Buttons

* Blizzard cooldown spiral
* Global Cooldown (GCD) support
* Range indicator
* Out of Mana indicator
* Item cooldowns
* Spell cooldowns
* Stack count display
* Hotkey display
* Tooltips

### User Interface

* Minimap button
* Context menu for each bar
* English user interface
* Lightweight configuration mode

### Performance

* Event-driven updates
* Optimized range checking
* Low CPU usage
* Designed for high FPS gameplay

## Supported Clients

* Turtle WoW
* Vanilla World of Warcraft 1.12
* Other compatible Vanilla-based projects

## Installation

1. Download the latest release.
2. Extract the folder into:

Interface/AddOns/

3. Restart the game or reload your UI.
